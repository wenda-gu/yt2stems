# Code of Conduct

## Our pledge

We want this project to be a welcoming, respectful place for contributors of all backgrounds and experience levels.

## Our standards

Examples of behavior that contributes to a positive environment:

- showing empathy and patience
- giving constructive feedback
- assuming good intent while staying honest about technical concerns
- focusing on what is best for the project and its users

Examples of unacceptable behavior:

- harassment or discrimination
- personal attacks
- trolling or deliberately disruptive behavior
- publishing private information without permission

## Enforcement

Project maintainers may remove comments, commits, issues, or contributors that violate this code of conduct.

## Scope

This code of conduct applies in project spaces and in public spaces when someone is representing the project.
