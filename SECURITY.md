# Security Policy

## Reporting a vulnerability

Please do not open a public issue for security-sensitive problems.

Instead, contact the maintainers privately with:

- a clear description of the issue
- steps to reproduce it
- the impact you believe it has
- any suggested mitigation

## Supported versions

This project is still pre-1.0. Security fixes will generally target the latest commit on the default branch until a formal release policy is documented.
