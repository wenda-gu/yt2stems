# yt2stems v0.1.1

Homebrew bootstrap fix release.

## Highlights

- Fixed `yt2stems-setup` and `install-yt2stems.sh` so they resolve symlinked install locations correctly
- Restored the intended Homebrew installation flow by allowing the bootstrap scripts to find the packaged project source
- Added a documented Tsinghua PyPI mirror option for users in China when `pip` downloads are unstable through the default path

## Upgrade notes

- Homebrew users should upgrade to this release before running `yt2stems-setup`
- If Python package downloads are unreliable from your network, run setup with:
  `PIP_INDEX_URL=https://pypi.tuna.tsinghua.edu.cn/simple PIP_TRUSTED_HOST=pypi.tuna.tsinghua.edu.cn`
