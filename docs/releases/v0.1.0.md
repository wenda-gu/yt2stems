# yt2stems v0.1.0

First public release of `yt2stems`.

## Highlights

- Installable Python package with CLI entry points for `yt2stems`, `yt2stems-benchmark`, and `yt2stems-install`
- Bootstrap installer for creating a dedicated virtual environment with the pinned runtime stack
- YouTube download flow powered by `yt-dlp` with browser-cookie support and retry handling for intermittent bot checks
- Demucs stem splitting with install-time default selection, clearer runtime messaging, and progress reporting
- Cleaner final output layout with `metadata.txt`, preserved `source.webm`, and no leaked raw `source.info.json` or `.demucs.log`
- Test suite, CI, contributor docs, and release automation for public maintenance

## Known limitations

- YouTube authentication behavior can still vary by browser profile, cookie state, and current `yt-dlp` compatibility
- Stem quality and speed still depend on the selected Demucs model and local hardware
- This release is tuned primarily for macOS and Apple Silicon workflows

## Upgrade notes

- Fresh installs should use `./yt2stems-setup`
- Existing private/local installs may need a reinstall to pick up the packaged public workflow and pinned dependencies
